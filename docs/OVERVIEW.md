# Overview

Telegram Knowledge Bot is a **personal knowledge capture system** built on top of Telegram.

It allows you to save anything you want to read, watch, or reference later â€” without breaking your flow.

Instead of bookmarks scattered across browsers, notes apps, and chats, everything lives in **one searchable Telegram inbox**.

---

## What is a Resource?

A resource can be:
- Article
- YouTube video
- Book
- Tutorial
- Blog post
- GitHub repo

Each resource stores:
- Title
- URL
- Type
- Status
- Tags
- Notes
- Timestamp
